Snap the Shot
=============

Snap the Shot predstavlja edukativnu igru za mobilne telefone sa Android operativnim sistemom.
Cilj u igri je da, na osnovu slike, igrač pronađe znamenitost u nekom gradu i fotografiše je.
Ukoliko je igrač fotografisao traženu znamenitost kao nagradu dobija opis te znamenitosti i poene kojima može da otključa neki od sledećih gradova.


INSTALL
=============

NetBeans plugin http://www.nbandroid.org/

Pre builda:
android update lib-project --path {putanja/do/projekta}/libs/google-play-services_lib


Podešavanja res/values/settings.xml

Podesiti adresu i port servera. 
Možete aktivirati test nivoe postavljanjem atibuta test_level na true.
